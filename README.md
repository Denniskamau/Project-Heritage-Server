# Project Heritage API

---

> A server for Project Heritage application

## Prerequisites

- Python v 3.6
- SQLite 3
- Python virtaulenv
- pip
- graphql

## Installation

```bash
create virtualenv
source venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
